# wedding-tv.cn AdSense 审核改进记录

**日期：** 2026-06-22  
**当前拒绝原因：** 低价值内容  
**处理策略：** 收窄审核入口，突出原创工具/指南，降低自动聚合内容和域名出售页权重。

## 已处理

1. **首页定位调整**
   - 将首页 title、description、OG/Twitter 信息从“域名出售”改为“免费婚礼工具、婚礼视频直播指南与各地婚俗大全”。
   - 将首屏 H1 和 CTA 改为工具/指南入口，域名出售不再作为首屏主诉求。
   - 将“为什么这个域名值得拥有”改为“本站内容价值”，突出免费工具、原创长文、婚俗大全、预算与流程参考。
   - 移除首页 FAQ、联系表单、浮动按钮和 JSON-LD 中的域名交易/报价语义，改为工具反馈、内容纠错和合作联系。

2. **低价值/聚合内容降权**
   - `news/` 下自动资讯页全部改为 `noindex,follow`。
   - `news/index.html`、`insights/index.html`、`blog/cities/index.html` 改为 `noindex,follow`。
   - 从这些高风险页面移除 AdSense 脚本，避免审核时广告代码出现在聚合/薄索引页面。

3. **政策与可信度页面**
   - `about.html`、`privacy.html`、`terms.html` 从 `noindex,follow` 改为 `index,follow,max-snippet:-1`。
   - 从政策页移除 AdSense 脚本，保留其作为可信度与合规说明页面。
   - 将这三页加入 `sitemap.xml`。
   - 将关于页、隐私政策和服务条款从域名交易说明改为网站工具、内容参考、反馈处理、版权和免责声明说明。

4. **Sitemap 清理**
   - 从 `sitemap.xml` 移除 `news/`、`insights/` 索引和 `blog/cities/` 薄索引入口。
   - sitemap URL 数量从 110 收窄到 59，保留首页、核心工具、指南、地域婚俗长文、深度洞察文章和政策页。

## 复审前建议

1. 部署后等待 Google 重新抓取 sitemap 和首页。
2. 在 AdSense 后台勾选“我确认已解决相关问题”后再申请审核。
3. 审核期间不要恢复自动新闻页索引，也不要在 `news/` 页面放置广告代码。
4. 若仍被拒，优先补强 5-8 篇人工编辑的婚礼视频/直播/预算原创指南，再提交。
